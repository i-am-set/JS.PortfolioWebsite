Privacy & Consent

This portfolio respects your privacy. It is hosted on a self-managed server and does not use third-party tracking scripts like Google Analytics or Meta Pixels.

Data Collection: The only tracking implemented is a custom, self-hosted view counter. This counter only increments if you explicitly grant "Analytics" consent via the cookie banner.

Cookies:

  • Necessary: A single cookie (portfolio_consent) is used to remember your consent preferences. This is required to prevent the banner from showing on every visit.
  • Analytics: If enabled, allows the site to send an anonymous POST request to a self-hosted API to increment a page view counter. No personal data or IP addresses are stored.
